Running an Arduino on a breadboard with NO EXTERNAL COMPONENTS / How to FLASH an Arduino BOOTLOADER

In this video, we discuss how to run an Arduino on a breadboard with no external components, and how to flash an Arduino bootloader onto an ATmega chip.
